Saya akan mengembangkan aplikasi web bernama "Aplikasi Survei Kepuasan Masyarakat (IKM)" versi 2.0.0 untuk instansi pemerintah.

Prompt 4.3 — Queue System (F-24) + Kalkulasi IKM (F-06)
text

Berdasarkan SRS F-06, F-24, dan bagian 7 (Algoritma Kalkulasi IKM), implementasikan:

1. Queue System berbasis Redis:
   - CodeIgniter 4 Queue Library atau custom dengan predis/predis
   - Queue: 'ikm-calculation', 'notification', 'backup', 'report', 'cleanup'
   - Worker: daemon yang berjalan terus menerus

2. Kalkulasi IKM:
   - Job: KalkulasiIKMJob menerima id_periode
   - Hitung NRR per unsur
   - Hitung NRR Tertimbang (× 0.111)
   - Hitung IKM Total = Σ NRR_t × 25
   - Tentukan mutu layanan: A/B/C/D
   - Simpan di tb_rekap_ikm
   - Hitung delta IKM vs periode sebelumnya
   - Set flag_alert jika penurunan > threshold (default 5 poin)
   - Trigger notification job jika alert

3. Dashboard Queue Monitor:
   - Jumlah pending, processing, failed, completed
   - Worker status (active, uptime, memory)
   - Retry failed jobs
   - Pause/resume queue

Buat:
- QueueService dengan Redis
- KalkulasiIKMJob
- QueueController untuk monitoring
- Command: queue:work, queue:retry, queue:monitor

Berikan kode lengkap.
